Parkinson Expert System Using Bayes' Theorem
Description
Parkinson Expert System is a web-based application developed to assist in the early detection of Parkinson's disease using Bayes' Theorem. The system allows users to select symptoms they experience and provide confidence levels for each symptom. Based on the selected symptoms and the knowledge base, the system calculates the probability of each disease category and provides the most likely diagnosis.

This application is intended to serve as a decision-support tool for early detection and does not replace professional medical diagnosis.
